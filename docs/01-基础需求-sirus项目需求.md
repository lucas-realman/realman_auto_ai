| **睿尔曼CRM项目需求清单** |
| **一级功能** | **二级功能** | **三级功能** | **功能说明** |
| **基础数据管理** | 基础数据管理 | 组织架构 | 支持独立设置组织架构，或者通过第三方系统对接获取组织架构 |
| 用户 | 支持独立设置用户，或者通过第三方系统对接获取用户，并为用户设置角色和权限 |
| 产品 | 支持手动创建或者对接ERP，同步产品信息 |
| **目标管理** | 目标管理 | 目标规则设定 | 支持对人员、客户、产品等维度设置目标，定义【什么组织人员】在【什么时间】完成【什么指标】的规则，支持子目标的拆分，灵活的规则定义 |
| 目标值设置 | 支持定义部门/人员的目标值，可按时间周期、人、客户、产品、部门的维度设定目标值，支持目标值导入。  例如：基于现有维度，设置订单、发货、回款的指标 |
| 目标完成情况 | 可按部门、人、时间、产品、客户等维度查看目标完成情况 |
| **线索管理** | 市场及线索管理 | 市场活动 | 录入市场活动，例如展会、活动，活动基本信息 |
| 市场线索 | 基于市场活动归集市场线索，并跟踪线索转化情况 |
| 新建线索 | 新建线索：支持直接新建线索，实时编辑相关信息，快速的将线索信息录入到CRM系统中 |
| 批量导入：支持基于excel模板批量导入线索 |
| 二维码自动收集：支持生成表单二维码，扫码自动收集表单信息为线索 |
| 移动端扫名片录入：支持移动端快速扫描名片，直接创建客户和联系人 |
| 线索池 | 可以新建多个线索池，更好的管理线索资源，通过设定分配、领取等规则，让线索高效流转 |
| 线索分配 | 支持自动分配和手动分配 |
| 手动领取：领取线索池中的业务数据，领取后线索归属于员工自己，员工可通过跟进界定线索质量从而进行后续流转。 |
| 线索回收 | 支持手动收回、手动退回 |
| 线索ROI分析 | 销售线索可以关联市场活动，通过线索转化分析市场活动投放效果，帮助企业精准较正市场营销策略 |
| **客户管理** | 客户信息管理 | 客户新建 | 1.支持新客户单条和批量导入方式直接创建客户，新建客户按照预配置字段填写对应信息  2.支持由线索转换为客户 |
| 客户主数据管理 | 1. 客户查重与合并  2. 客户动态数据记录（支持动态碎片化更新客户信息，以时间轴形式展现客户动态）  3. 客户分类分级，例如基于行业、区域、客户体量进行分类分级，支持动态更新评级  4. 支持客户标签  5. 客户负责人变更和交接  6. 客户信息编辑与历史 |
| （国内）客户工商信息联动 | 工商联动：支持对接客户工商信息，校验工商信息并回填工商信息到客户资料中 |
| 客户邮件归档 | 支持绑定邮箱，一键将重要邮件归档到客户资料下，方便邮件类型资料的客户档案归档 |
| 客户360视图 | 基于客户建立基本信息、联系人、竞争对手、商机、拜访记录、合同订单、发货、回款开票、工单等全面信息的360度视图 |
| **联系人管理** | 联系人新建 | 联系人新建 | 支持新建客户、支持线索转客户同时新建联系人，联系人基本信息：包括姓名、部门、联系方式、地址等 |
| 联系人图谱 | 联系人图谱 | 可以基于客户联系人基本信息及联系人之间的关系构建可视化图谱，实现决策链条分析 |
| **商机管理** | 商机报备和立项 | 商机报备及准入机制 | 建立商机报备机制，完成商机准入的过程，基于商机进行报备查重、审核，激励销售团队及早提报商机，避免直营和渠道间的冲突 |
| 商机团队 | 商机团队 | 基于组织架构、部门角色，快速建立团队（销售、商务、技术支持），快速拉通项目售前协作 |
| 商机推进 | 项目流程和任务 | 支持不同类型的商机，例如标品商机、定制项目商机，为不同商机规划不同的销售路径支持不同销售组织的不同销售过程  基于过程配置关键阶段和任务，为任务分配对应的人员和要求 |
| 商机结果及复盘 | 商机无效、丢单原因填写；商机赢单总结与丢单分析 |
| 漏斗分析 | 基于商机阶段建立商机可视化漏斗，直观分析销售转化情况 |
| 商机360视图 | 基于商机可配置建立基本信息、商机决策链、竞争对手、招投标、报价、拜访记录、合同订单等全面信息的360度视图 |
| 商机看板 | 通过运营看板实现项目群状态监控、风险预警、各阶段赢单率分析等 |
| 报价 | 产品价格配置 | 为产品建立不同的价格策略，例如直销和渠道的差异化价格 |
| 报价单审批 | 基于报价单的价格，触发差异化的审批流程（例如不同折扣对应不同的审批层级） |
| 报价单打印 | 报价单打印：支持配置报价单打印模板，自动生成打印模板 |
| 样机管理 | 样机管理 | 支持样机的申请、借用、归还、转订单的闭环管理 |
| 投标管理 | 投标管理 | 支持投标流程的管理：招标信息录入、投标申请、保证金申请、开标结果复盘，实现投标闭环规范化管理 |
| 商机结果及复盘 | 商机结果及复盘 | 商机无效、丢单原因填写；商机赢单总结与丢单分析 |
| **产品管理** | 产品 | 产品信息 | 产品信息由ERP同步至CRM，包含产品名称、产品规格、参数等 |
| 价目表 | 价目表 | 对产品指定售卖指导价，方便销售报价及下单时做调取选用 |
| **合同管理** | 框架合同 | 框架合同 | 支持框架合同，录入合同基本信息、附件，提交后触发对应审批流程。基于不同合同类型，触发技术、财务、管理等角色进行评审，审批通过后合同生效  支持基于框架合同约束合同内的产品和价格。 |
| 标准合同（订单） | 标准合同（订单） | 支持标准合同订单的创建、基于合同条款，触发技术、财务、管理等角色进行评审，审批通过后合同生效 |
| 提货单录入 | 提货单录入 | 支持基于框架合同录入提货单，进行台账记录。 |
| **订单管理** | 订单创建 | 订单创建 | 提供订单创建功能，录入订单相关信息，并提供系统自动校验和提醒，提高订单录入准确率和录入效率。  支持报价单 ( PI ) 转销售订单 (SO)  订单审批后生成正式订单，自动同步到ERP |
| 订单跟踪 | 订单跟踪 | 在CRM侧查询订单执行的状态（由ERP返回关键状态信息） |
| 发货申请 | 发货申请 | 在CRM侧可以基于订单发起发货申请：选择相应的订单行，填写发运数量进行发货申请提交（支持部分发货）；  发货申请同步到ERP，ERP发货后，将关联的发货单回传CRM。 |
| 物流跟踪 | 物流跟踪 | 支持对接物流，实时跟踪发货后的物流进度和签收进度。 |
| **回款和发票** | 回款管理 | 流水同步 | 回款数据清单从CBS同步至CRM |
| 回款认领 | 在CRM侧做回款数据的认领及核销 |
| 回款数据同步 | 认领核销后的回款数据同步至ERP |
| 开票管理 | 开票申请 | 在销售侧管理开票基本信息，基于合同、订单维度提交开票申请。 |
| 开票台账 | 同步ERP数据：ERP将开票记录/发票链接或文件同步到CRM，并自动关联到对应的订单下，方便业务人员在CRM中查看开票情况 |
| **退货退款** | 退货申请 | 退货申请 | 业务侧在线创建退货申请单，关联所属客户、所属订单，退货原因、以及需要退货的商品规格、数量等，希望退货完成时间等内容，字段可自定义，发起申请后，审批流程自定义，审批过后同步至U9C系统 |
| 退款管理 | 退款申请 | 在线发起退款申请，关联所属客户、所属订单、退货申请单、退款金额、客户方收款账号、计划退款时间等内容，发起后审批流程自定义，审批过后将退款单据同步至U9c |
| 退款结果 | 财务方完成退款后，将退款状态同步至CRM，方便前端业务了解进度 |
| **工单管理** | 工单类型 | 工单类型 | 支持售前工单、安装工单、售后工单等不同的工单类型，为不同类型工单创建不同的业务流程 |
| 工单创建 | 工单创建 | 支持销售、内勤、售后等内部用户为客户创建工单 |
| 工单执行 | 工单执行 | 支持为工单创建业务流程，分配对应的负责人和任务 |
| 工单关闭 | 工单关闭 | 工单完成后，录入对应的记录，进行工单关闭 |
| **行为管理** | 销售记录 | 销售记录 | 支持快速录入或者语音填写销售记录，支持记录关联到客户 |
| 外勤拜访 | 外勤拜访 | 创建外勤计划：员工提交拜访计划，关联拜访联系人，关联客户和商机，可基于计划配置审批  执行拜访计划：基于位置信息实时打卡，将打卡信息自动记录到客户销售记录中，碎片化沉淀客户资料  自动生成记录：员工执行拜访计划后，自动将拜访记录到对应关联的模块下，同时可基于拜访情况生成拜访统计 |
| 日报周报 | 日报周报 | 支持销售人员填写日报/周报，设置未来的周计划、总结上周完成情况。  上级领导可对日报周报进行点评 |
| **数据分析** | 数据报表 | 数据报表 | BI数据报表：客户对系统内的所有报表数据进行各类分析，支持表格和图标形式（例如基于数据形成销售画像报表） |
| 统计图 | 统计图 | 图形化展现数据，直观查看数据统计结果，协助企业实时掌握经营状态，打造以数据为中心的实时决策 |
| 图表订阅 | 图表订阅 | 关注特定图表的数据，及时洞察数据情况，支持企信和邮箱订阅 |
| 数据驾驶舱 | 数据驾驶舱 | 基于BI报表和统计图的能力，汇总企业运行的关键指标，直观的监测企业运营情况 |
| 报表导出 | 报表导出 | 支持基于生成的报表导出excel，统计图可导出对应图片 |
| **系统集成** | 钉钉 | 钉钉 | 组织/人员由钉钉同步至CRM |
| 消息待办推送：CRM将待办信息推送到钉钉。钉钉侧收到销售后，无需登录直接打开CRM待办内容页面进行业务处理 |
| ERP(U9) | ERP(U9) | 产品同步：产品档案同步到CRM作为产品 |
| 汇率：ERP将汇率同步至CRM |
| 客户：CRM将成交客户(联系人）信息推送到ERP |
| 销售订单：销售订单由CRM同步至ERP |
| 发货单申请：发货单申请由CRM同步至ERP |
| 出库单：ERP将出库单同步至CRM，用于及时提醒销售出库状态 |
| 回款单：CRM将回款单同步至ERP |
| 发票申请：CRM中发起发票申请信息同步至ERP |
| 发票结果：ERP将发票信息同步至CRM |
| 退货：CRM将退货单信息推送到ERP |
| 退款：CRM将退款单信息推送到ERP，退款结果由ERP同步至CRM |
| 退款结果：结果状态由U9同步至CRM |
| 分贝通 | 分贝通 | 客户：CRM将客户信息推送到分贝通 |
| 商机：CRM将商机信息推送到分贝通 |
| MES | MES | 将生产进度同步至CRM |
| **多语言及国际地图** | 国际地图 | 国际地图 | 支持国际地图，实现海外客户地图定位和签到打卡 |
| 多语言 | 常见语言 | 支持多种语言（中英双语）  支持自定义语种  提供翻译工作台 |
| **AI智能交互** | 自然语言交互 | 自然语言查询 | 支持用户通过自然语言输入查询CRM数据，例如"查看本月新增客户"、"张三的商机进展如何"等，系统自动解析意图并返回结果 |
| 自然语言操作 | 支持通过自然语言指令完成常用CRM操作，例如"新建一个线索，客户名称XXX"、"将商机阶段推进到方案报价"等，降低系统使用门槛 |
| 多轮对话 | 支持上下文关联的多轮对话，用户可在对话中逐步细化查询条件或操作指令，系统能理解上下文进行连续交互 |
| 钉钉交互入口 | 钉钉机器人 | 在钉钉中部署CRM智能助手机器人，用户可在钉钉单聊或群聊中通过@机器人发送自然语言指令，直接与CRM系统交互，无需切换系统 |
| 钉钉消息卡片 | 机器人返回的查询结果、审批通知、待办提醒等以钉钉互动卡片形式展示，支持在卡片内直接操作（如审批、确认、跳转详情） |
| 钉钉工作台集成 | 在钉钉工作台提供CRM快捷入口，支持从钉钉直接访问CRM核心功能页面 |
| AI智能总结 | 客户总结 | 基于客户360视图数据，AI自动生成客户画像摘要，包括客户基本情况、历史交易、近期动态、风险提示等，帮助销售快速了解客户全貌 |
| 商机总结 | AI自动分析商机全流程数据，生成商机推进摘要，包括当前阶段、关键事件、风险点、下一步建议等 |
| 会议/拜访总结 | 根据销售录入的拜访记录或会议纪要，AI自动提取关键信息并生成结构化总结，自动关联到对应客户和商机 |
| 周报/月报自动生成 | 基于销售人员的工作数据（拜访、商机推进、订单、回款等），AI自动生成周报/月报草稿，销售确认后即可提交 |
| 数据洞察总结 | AI自动分析报表数据，生成关键指标变化趋势、异常波动提醒和业务建议的文字总结 |
| **钉钉权限管理** | 钉钉统一身份 | 钉钉SSO单点登录 | 通过钉钉实现统一身份认证，用户使用钉钉账号登录CRM系统，无需单独注册和维护CRM账号密码 |
| 组织架构同步 | 自动同步钉钉组织架构和人员信息到CRM，人员入职、离职、调岗在钉钉操作后自动同步至CRM，确保权限实时更新 |
| 角色权限映射 | 基于钉钉组织架构和角色体系，自动映射CRM中的数据权限和功能权限，例如部门主管自动获得本部门数据查看权限 |
| 权限变更联动 | 当钉钉侧发生人员变动（离职、转岗等），CRM权限自动同步调整，确保数据安全，离职人员权限即时回收 |
